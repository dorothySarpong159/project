# PHP PROJECT - PHASE 1
# FunDee Collection
A morden fashion company created for our PHP Projrct which displays a collection of clothes in a trendy era for users to purchase and shop

# Technologies
- HTML
- CSS
- PHP
- Google fonts
- MySQL
- Version Control via Github

  # Project Purpose
- Homepage with carousel and feature cards as the first introduction to the company
- About page which contains the purpose of the company
- Shop page which previews the product avilable
- Individual product pages for users to explore more about the product
- Contact page for users in need of help with products
- Login and register pages for users to be able to easily shop with an account

# Setup Instruction
To view the project so far:
- Download the repository from GitHub
- Upload in a code editor eg Visual Studio Code
- View the pages on your local server(XAMPP/LAMPSTACK)
- Database included but not yet connected
  To explore the pages on you browser:
  - index.php - homepage
  - about.php - about page
  - product-1.php - one of the product pages(there are 6 product pages)
  - contact.php - contact form
  - login.php/register.php - account pages
